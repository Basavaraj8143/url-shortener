## Day 1

Created standard Spring Boot package structure.

Created first domain model:

Url.java

Fields:
- id
- longUrl
- shortCode
- createdAt

Concept Learned:
Entity / Domain Model

Understanding:
The Url class represents the core business object of the application and will later be mapped to a database table.


## Day 1.5

Converted Url.java into a JPA Entity.

Annotations Learned:

- @Entity-- this should be stored in db 
- @Table-- Create/use table named urls
- @Id --PRIMARY KEY
- @GeneratedValue(strategy = GenerationType.IDENTITY)--Database generates id automatically.

Understanding:

JPA maps Java classes to database tables.

Url object ↔ urls table

Primary keys can be auto-generated by the database using IDENTITY strategy.
